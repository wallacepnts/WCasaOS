Atualizar
~~~
sudo apt update && sudo apt upgrade
~~~

Instalar curl
~~~
sudo apt install curl
~~~

Instalar o Casa OS
~~~
curl -fsSL https://get.casaos.io | sudo bash
~~~

Adicionar a fonte na loja de aplicativos
~~~
https://github.com/wallacepnts/WCasaOS/archive/refs/heads/main.zip
~~~
